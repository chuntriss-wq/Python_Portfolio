# Text Battle RPG

A simple turn-based battle game built with JavaScript.

## How to Play

1. Open `index.html` in your web browser
2. Click "ATTACK" to fight the enemy
3. Battle continues until either you or the enemy reaches 0 HP
4. Click "Play Again" to restart

## Features

- Turn-based combat system
- Random damage calculation (5 to max attack power)
- Critical hit system (15% chance for double damage)
- Health tracking for player and enemy
- Battle log showing all actions
